# Changelog — LFB Sales Leads Portal

## 2026-03-21 — Initial Release

- Created sales leads portal with Supabase backend
- Email/password authentication (Supabase Auth)
- Three pipeline stages: Scouting, Hot Lead, Converted
- Segment tabs for quick filtering by status
- Full-text search across name, company, email, phone, status
- Sortable columns (click any header)
- Pagination with per-page selector (25/50/100/All)
- Add, edit, delete leads with modal form
- CSV export of filtered results
- Real-time sync across browsers/devices
- Clickable stat cards for quick filtering
- Purple gradient header (distinct from wholesale portal)
- Connection status indicator
- Loading bar and toast notifications
- Mobile responsive design
- GitHub Pages deployment with PUSH_TO_SALES_LIVE.bat
- Auto-push watcher for hands-free deployment
